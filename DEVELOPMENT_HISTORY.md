# Development History

`v0.1.17 final` is the first public-ready packaged release of Spiky Batch Macro.

Development progressed from safe table detection and single-sample peak analysis to validated baseline-anchor extraction, polynomial baseline correction, corrected-trace peak analysis, recoverable Full Batch processing, aggregate tables, a lightweight workbook, conservative baseline QC, provenance checks, source-aware filenames, and organized outputs.

Maintenance validation used a deterministic 96-well synthetic stress test. Reporting and workbook runtime were optimized without changing scientific calculations. The public-release pass added licensing, attribution, governance, synthetic examples, and repository/package sanitization.
